Manejo de arreglos

Como utilizar los métodos de arreglo push, pop, shift y unshift en JavaScript

Números primos

        
            let primos = [];
            for (let i = 2; i <= 10000000; i++) {
                let esPrimo = true;
                for (let j = 2; j <= i - 1; j++) {
                    let modulo = i % j;
                    if (modulo == 0) {
                        esPrimo = false;
                        break;
                    };
                };
                if (esPrimo) {
                    primos.push(i);
                };
            };
